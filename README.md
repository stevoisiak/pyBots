# pyBots

## telegram

Telegram BOT made in python that shows random images when it's called.

### requirements
- [telepot](https://github.com/nickoala/telepot) → `pip install telepot` or `python3 setup.py install`
- requests → `pip install requests`
- bs4 → `pip install bs4`
